# To-Do List Application

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**. This project helps users organize their daily tasks efficiently with a clean and user-friendly interface.

## Features

- Add new tasks easily.
- Delete tasks when completed.
- Mark tasks as done.
- Responsive and smooth user interface.
- Organized layout for better productivity.
